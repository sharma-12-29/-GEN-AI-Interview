# AI Interview Preparation Platform

An AI-powered interview preparation platform built using the MERN Stack. The platform analyzes a user's profile according to a target job description and generates a personalized preparation strategy, learning roadmap, and AI-based performance report.

## Features
- Job description analysis
- Personalized interview preparation roadmap
- Skill gap analysis
- AI-generated preparation strategy
- Resume/profile matching with job roles
- Performance and readiness reports
- Responsive modern UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Gemini/OpenAI API

## Installation

```bash
git clone https://github.com/your-username/GEN-AI-Interview.git
cd GEN-AI-Interview
npm install
npm run dev
```
